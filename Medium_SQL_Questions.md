# Medium SQL Interview Questions

## Question 1: Find Top 3 Highest Salaries

### Query

```sql id="lvmnq1"
SELECT *
FROM Employees
ORDER BY Salary DESC
LIMIT 3;
```

### Explanation

Sorts employees by salary in descending order and returns the top 3.

---

## Question 2: Find Revenue by City

### Query

```sql id="qhx2zj"
SELECT
City,
SUM(Revenue) AS Total_Revenue
FROM Orders
GROUP BY City;
```

### Explanation

Calculates total revenue generated from each city.

---

## Question 3: Find Customers With More Than 5 Orders

### Query

```sql id="c81vxn"
SELECT
Customer_ID,
COUNT(Order_ID) AS Total_Orders
FROM Orders
GROUP BY Customer_ID
HAVING COUNT(Order_ID) > 5;
```

### Explanation

Filters customers who have placed more than 5 orders.

---

## Question 4: Find Products Generating More Than ₹10,000 Revenue

### Query

```sql id="9m42fr"
SELECT
Product_Name,
SUM(Revenue) AS Total_Revenue
FROM Sales
GROUP BY Product_Name
HAVING SUM(Revenue) > 10000;
```

### Explanation

Returns products with revenue greater than ₹10,000.

---

## Question 5: Find Employees Earning Above Average Salary

### Query

```sql id="kq8wpr"
SELECT *
FROM Employees
WHERE Salary >
(
    SELECT AVG(Salary)
    FROM Employees
);
```

### Explanation

Uses a subquery to compare salaries against the company average.

---

## Question 6: Find Duplicate Records

### Query

```sql id="bvl6hr"
SELECT
Email,
COUNT(*) AS Duplicate_Count
FROM Customers
GROUP BY Email
HAVING COUNT(*) > 1;
```

### Explanation

Identifies duplicate email records.

---

## Question 7: Calculate Monthly Revenue

### Query

```sql id="uxtx0k"
SELECT
MONTH(Order_Date) AS Month_Number,
SUM(Revenue) AS Total_Revenue
FROM Orders
GROUP BY MONTH(Order_Date);
```

### Explanation

Calculates revenue month-wise.

---

## Question 8: Find Customers Who Never Ordered

### Query

```sql id="ycf7qn"
SELECT
c.Customer_ID,
c.Customer_Name
FROM Customers c
LEFT JOIN Orders o
ON c.Customer_ID = o.Customer_ID
WHERE o.Order_ID IS NULL;
```

### Explanation

Finds customers without any orders.

---

## Question 9: Find Second Highest Salary

### Query

```sql id="jdr87a"
SELECT MAX(Salary)
FROM Employees
WHERE Salary <
(
    SELECT MAX(Salary)
    FROM Employees
);
```

### Explanation

Finds the salary immediately below the highest salary.

---

## Question 10: Revenue Contribution by Category

### Query

```sql id="2eqym8"
SELECT
Category,
SUM(Revenue) AS Total_Revenue
FROM Products
GROUP BY Category
ORDER BY Total_Revenue DESC;
```

### Explanation

Calculates revenue generated by each product category.

## Bonus Question: Most Valuable Customer

### Query

```sql id="f83r5n"
SELECT
Customer_ID,
SUM(Revenue) AS Total_Spent
FROM Orders
GROUP BY Customer_ID
ORDER BY Total_Spent DESC
LIMIT 1;
```

### Explanation

Finds the customer with the highest total spending.
